# Portfolio

## Installation

Use
```bash
git clone https://github.com/gonzyui/portfolio.git
```

Then go into `portfolio` directory:

```bash
cd portfolio/
```

Then install dependencies and build the project:

```bash
npm install
## or
yarn add

npm run build
```
